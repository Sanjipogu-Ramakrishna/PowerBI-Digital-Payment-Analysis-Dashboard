# Market trends-Digital-Payment-Analysis (Interactive Dashoard creation using PowerBI)
## Project Objective
The business clients want to know where the highest digital payments were going in which categories reports for 2024.So that business client can unserstand where their customers spending their maximum amount.So this dashboard will  helps to the client to increase their branding promotion or their investmetns on a particulat category in future.

## Dataset used
- <a href="https://github.com/Sanjipogu-Ramakrishna/PowerBI-Digital-Payment-Analysis-Dashboard/blob/main/Digital%20Payment%20Data%20(P).xlsx">Dataset</a>

## Questions and (KPIs)

- Which payment mode is used most (UPI, Card, Net Banking)?

- Monthly transaction growth rate?

- What are the top 3 categories done highest transaction amount?

- among all the users separate the active users?

- Failed vs Successful transaction percentage?

- Average transaction value?
  
- Dashboard interaction <a href="https://github.com/Sanjipogu-Ramakrishna/PowerBI-Digital-Payment-Analysis-Dashboard/blob/main/Screenshot%202026-03-05%20140145.png">view Dashboard</a>

## Process
- Verify data for any missing values and anomalies, and sort out the same.
- Make sure data is consistent and clean with respect to the data type, data formate and values used.
- Created charts and table according to the questions asked.
- Merge all charts and table into one dashboard and apply slicers to make dynamic.
